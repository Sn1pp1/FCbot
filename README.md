## 📖 О проекте

**FC Bot** — это Telegram-бот для быстрой и удобной организации футбольных матчей. 

Забудьте про споры "кто с кем играет", долгие согласования и ручное деление на команды. Этот бот возьмет все организационные вопросы на себя!

### ✨ Почему стоит использовать FC Bot?

-  **Экономия времени** — организация матча за пару кликов
- 🎯 **Простота** — интуитивно понятный интерфейс
- ⚖️ **Честность** — автоматическая жеребьевка без предвзятости
- 📱 **Доступность** — работает в любом Telegram-чате

## 🚀 Возможности

| Функция | Описание |
|---------|----------|
| 📝 **Быстрый сбор** | Запись игроков на игру в один клик |
| 🎲 **Жеребьевка** | Автоматическое деление на равные команды |
| ⚡ **Минимум настроек** | Не нужно ничего настраивать — просто начните |
| 👥 **Учет игроков** | Автоматический подсчет участников |
|  **Статистика** | Отслеживание результатов матчей |
